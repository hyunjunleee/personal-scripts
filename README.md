# personal-scripts
Collection of automation scripts from my personal projects

평소 작업 과정에서 발생하는 병목과 파편화 문제를 해결하기 위해, 직접 제작하고 유지보수하는 **개인 유틸리티 스크립트 모음(Monorepo)**입니다.

##  Repository Overview
목적과 실행 환경에 따라 최적의 언어와 도구를 선택하여 문제를 해결합니다. 각 폴더를 클릭하면 상세한 기능과 사용법을 확인할 수 있습니다.
