# AutomatedChartPlotter (2019)

선택한 데이터 범위를 바탕으로 일관된 서식의 산점도(XY Scatter) 차트를 빠르게 생성해 주는 엑셀 VBA 매크로 모음입니다. 반복적인 차트 서식 지정 작업(축 제목 설정, 눈금선 제거, 추세선 및 수식 추가 등)을 자동화하여 작업 시간을 단축할 수 있습니다.

## 주요 기능 (Features)

이 모듈에는 두 가지 매크로가 포함되어 있습니다:

### 1\. chartmaker (오차막대 없는 차트)

* 선택한 셀 데이터를 기반으로 산점도(XY Scatter) 차트 생성
* 차트 서식 자동 적용:

  * 차트 제목("Title") 및 X/Y축 제목("X \[]", "Y \[]") 자동 생성 (폰트: Bahnschrift)
  * 가로/세로 기본 눈금선 제거로 깔끔한 배경 유지
  * 선형 추세선 추가 및 수식, R² 값 표시
  * 데이터 마커 서식 지정 (원형, 크기 4, 검은색 톤)

### 2\. chartmaker\_errb (오차막대 포함 차트)

* chartmaker의 모든 서식 지정 기능 포함
* 데이터 마커에 X축 및 Y축 표준 오차(Standard Error) 오차막대 자동 추가

## 단축키 (Shortcuts)

|매크로 이름|설명|단축키|
|-|-|-|
|chartmaker|기본 산점도 + 선형 추세선 생성|Ctrl + g|
|chartmaker\_errb|기본 산점도 + 선형 추세선 + 오차막대 생성|Ctrl + Shift + G|

*(참고: 엑셀 매크로 옵션에서 단축키가 정상적으로 할당되어 있는지 확인이 필요할 수 있습니다.)*

## 사용 방법 (How to Use)

1. 엑셀 시트에서 차트로 변환할 데이터 범위(X값과 Y값이 포함된 셀)를 드래그하여 선택 (Ctrl을 눌러 떨어져 있는 두 열 선택 가능).
2. 지정된 단축키(Ctrl + g 또는 Ctrl + Shift + G)를 누르거나, 보기 > 매크로 메뉴에서 원하는 매크로를 선택해 실행.
3. 생성된 차트의 "Title", "X \[]", "Y \[]" 텍스트를 클릭하여 원하는 제목으로 수정하여 사용.

## 사용자 지정 (Customization)

기본적으로 적용되는 폰트(Bahnschrift), 텍스트 내용, 마커 크기(MarkerSize = 4) 등을 변경하고 싶다면, VBA 편집기(Alt + F11)에서 코드를 열어 해당 속성값을 직접 수정하여 사용할 수 있습니다.
